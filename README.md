<h2>注意点</h2>
<h3>パスの問題</h3>
<p>
各動画とサンプルコードの対応を分かりやすくするため、ディレクトリ名に日本語使用しております。
実際に動かす際、パスに日本語が含まれていると正しく動作しない場合がありますので、
適宜アルファベットなどに置き換えてお試しください。
</p>
<h3>APIキー</h3>
<p>
サンプルコードによってはAPIキーを設定しないと動作しないものがあります。
適宜APIキーの取得設定をおこないお試しください。
</p>
